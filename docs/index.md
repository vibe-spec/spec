# .vibe Specification

**Make Your System READable** — an open, Markdown‑based contract language for AI‑driven development.

---

**.vibe** is a minimalist, AI‑first contract language built on Markdown. It uses just four core symbols (`!` `->` `?` `#`) to capture the essential dimensions of any software system: **Rules, Effects, Anomalies, and Decisions** – the **READ** principle.

### Why .vibe?

- **Zero learning curve** – if you know Markdown, you already know .vibe.
- **AI native** – symbols act as attention anchors; any LLM understands them immediately.
- **Universal** – works with every Vibe IDE (Cursor, Claude Code, Windsurf, …) and every model.
- **Progressive** – start with a single `!` and enrich only when needed.
- **Secure by design** – security requirements are first‑class citizens (`!security‑*`).
- **No lock‑in** – plain text, Git‑friendly, CC BY 4.0 licensed.

## CN

### 为什么选择 .vibe？

- **零学习成本** – 会写 Markdown 就会写 .vibe。

- **开放通用** – 兼容所有 Vibe IDE（Cursor、Claude Code、Windsurf……）和所有模型。
- **渐进增强** – 从一个 `!` 开始，按需丰富语义。
- **安全内生** – 安全需求是一等公民（`!security‑*`）。
- **无锁定** – 纯文本、Git 友好、采用 CC BY 4.0 许可证。
